Wexio Widget — CDN loader example

This page loads the Wexio widget via the script-injection loader.js. The widget mounts as a sandboxed iframe in the bottom-right corner; the loader handles postMessage events for open/close/resize.

View source to see the embed — it's a single <script> tag.

Pre-boot known users (optional)

Set window.wexioSettings.user BEFORE the loader runs to log a known visitor in: 

<script>
  window.wexioSettings = {
    user: {
      jwt: "<host-signed-jwt>",
      name: "Ada Lovelace",
      email: "ada@example.com",
    },
  };
</script>
<script async src="https://cdn.wexio.io/widget/loader.js" data-public-key="pk_live_..."></script>

Full docs: learn.wexio.io/docs/web-widget
